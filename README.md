# devops-essentials
Image with All essentials libraries for DevOps Tool 
Following are the list of libraries added to alpine image :
- bash
- openssl
- python3
- terrafrom
- git
- curl
- kubectl
- helm "helm3"
- kustomize

# Docker Commands 
   docker build -t tiwarisanjay/devops-essentials:2.0.0 --build-arg KUSTOMIZE_VERSION=4.4.1 .
